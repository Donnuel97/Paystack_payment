How many ways can you integrate Paystack into your app?

There are three ways of integrating PAYSTACK into you app, two of which are paystack inline and paystack standard, for this presentation I used paystack inline.

Did you encounter any issues when using the Paystack APIs, and how did you resolve them?

while working with the API, i had some initial issue with the way the values where represented, that is if I put 300, on the API it would come out as 3, to solve this problem, before saving an input, I multiplied it by 100.
